Auth

Sending auth paramters

Sent to authenticate user

Direction Client
Message A|REQ|loginData+
Ack A|A+

Invalid Auth

Recieved if login fails

Direction Server
Message A|E|INVALID_AUTH_DATA|unknown user+

Too many auth attempts

Recieved if login fails more than the maximum allowed attempts and closes the client connection

Direction Server
Message A|E|TOO_MANY_AUTH_ATTEMPTS+

Invalid Message

Direction
Message X|E|INVALID_MESSAGE_DATA|invalidMessage+

Unknown Action

Direction
Message X|E|UNKNOWN_ACTION|unknown action actionName+

Events

Listen

Send when listening to event matching regex

Direction Client
Message E|L|event\/.*+
Ack E|A|L|event\/.*+

Unlisten

Send when no longer interested in listening to event

Direction Client
Message E|UL|event\/.*+
Ack E|A|UL|event\/.*+

Susbcribe

Send to subscribe to event

Direction Client
Message E|S|eventName+
Ack E|A|S|eventName+

Unsubscribe

Send to unsubscribe to event

Direction Client
Message E|US|eventName+
Ack E|A|US|eventName+

Publish

Recieved/Sent when an event occurs

Direction Client/Server
Message E|EVT|eventName|JSON+
Ack E|A|EVT|eventName+

Listen Match

Recieved when a listened to regex is matched

Direction Server
Message E|SP|event\/.*|event/A+

Invalid Message

Direction
Message E|E|INVALID_MESSAGE_DATA|invalidMessage+

Unknown Action

Direction
Message E|E|UNKNOWN_ACTION|unknown action actionName+

RPC

Register as provider

Sent when client can recieve RPC events

Direction Client
Message P|S|rpcName+
Ack P|A|S|rpcName+

Unregister as provider

Sent when client no longer wishes to recieve RPC events

Direction Client
Message P|US|rpcName+
Ack P|A|US|rpcName+

Send RPC request

Send/Recieve when RPC requested

Direction Client/Server
Message P|REQ|rpcName|rpcID|Arguments+
Ack P|A|REQ|rpcName|rpcID+

Reject RPC request

Send if client does not want to execute RPC

Direction Client
Message P|REJ|rpcName|rpcID+

Error RPC request

Direction Send if error occurs during execution
Message P|E|ErrorMessage|rpcName|123+

Respond to RPC request

Direction Send if error occurs when trying to execute rpc
Message P|RES|rpcName|rpcID|Result+

Invalid Message

Direction
Message P|E|INVALID_MESSAGE_DATA|invalidMessage+

Unknown Action

Direction
Message P|E|UNKNOWN_ACTION|unknown action actionName+
