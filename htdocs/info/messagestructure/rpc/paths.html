RPC

happy path

Registers and unregisters an RPC

Direction Client Action Message
Out ClientA Register as provider P|S|rpcName+
In ClientA Register as provider Ack P|A|S|rpcName+
Out ClientA Unregister as provider P|US|rpcName+
In ClientA Unregister as provider Ack P|A|US|rpcName+
happy path

Gets a request and processes it succesfully

Direction Client Action Message
Out ClientB Send RPC request P|REQ|rpcName|rpcID|Arguments+
In ClientB Send RPC request Ack P|A|REQ|rpcName|rpcID+
In ClientA Send RPC request P|REQ|rpcName|rpcID|Arguments+
In ClientA Send RPC request Ack P|A|REQ|rpcName|rpcID+
Out ClientA Respond to RPC request P|RES|rpcName|rpcID|Result+
In ClientB Respond to RPC request P|RES|rpcName|rpcID|Result+
unhappy path

Rejects an RPC

Direction Client Action Message
Out ClientB Send RPC request P|REQ|rpcName|rpcID|Arguments+
In ClientB Send RPC request Ack P|A|REQ|rpcName|rpcID+
In ClientA Send RPC request P|REQ|rpcName|rpcID|Arguments+
Out ClientA Reject RPC request P|REJ|rpcName|rpcID+
In ClientB Send RPC request P|REQ|rpcName|rpcID|Arguments+
Out ClientB Respond to RPC request P|RES|rpcName|rpcID|Result+
In ClientB Respond to RPC request P|RES|rpcName|rpcID|Result+
unhappy path

Notifies client RPC errored during execution

Direction Client Action Message
Out ClientB Send RPC request P|REQ|rpcName|rpcID|Arguments+
In ClientB Send RPC request Ack P|A|REQ|rpcName|rpcID+
In ClientA Send RPC request P|REQ|rpcName|rpcID|Arguments+
Out ClientA Error RPC request P|E|ErrorMessage|rpcName|123+
In ClientB Error RPC request P|E|ErrorMessage|rpcName|123+
